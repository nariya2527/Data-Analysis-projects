# Student-Performance-Analysis

## 📊 Student Performance Analysis

This project focuses on analyzing student performance data using Python.  
The objective is to identify trends, compare subject-wise performance, and derive meaningful insights.

### 🔧 Tools & Technologies:
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

### 📈 Key Analysis:
- Compared student performance across different subjects  
- Identified highest and lowest scoring subjects  
- Visualized trends using bar charts, line plots, and pie charts  

### 📊 Visualizations:
- Bar Chart  
- Line Plot  
- Pie Chart  

### 💡 Key Insights:
- Certain subjects showed consistently higher performance  
- Variation in scores helped identify weak areas  
- Data visualization made performance comparison easier
